# Editorial Calendar Template

## Instructions

**Purpose**: This template provides a structured way to plan, organize, and track your content production. It helps you visualize your content pipeline and ensure a consistent publishing schedule.

**How to Use (Pro Tier - Interactive)**: This is a Markdown file. While you can use it as is, it's most effective when converted into a spreadsheet (Google Sheets, Excel, Airtable). You can copy and paste the table below into your preferred tool.

### **Column Breakdown**

*   **Publish Date**: The target date for the content to go live.
*   **Working Title**: The current title of the article.
*   **Author**: The person responsible for writing the content.
*   **Status**: The current stage of the content in the workflow (e.g., Idea, Outlined, In Progress, In Review, Scheduled, Published).
*   **Pillar**: The main pillar this content supports.
*   **Cluster**: The topic cluster this content belongs to.
*   **Primary Keyword**: The main keyword you are targeting.
*   **CTA**: The primary call-to-action for the article.
*   **Notes**: Any other relevant information, such as target audience, key stats to include, or promotion plans.
